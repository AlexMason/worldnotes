**The 5 assertions to preserve (exact expected values):**

1. **"converts br elements to newline characters"**
   - Input: `element('DIV', [text('first'), element('BR'), text('second')])`
   - Call: `getTextOffset(div, null, 0).text`
   - Expected: `'first\nsecond'`

2. **"preserves line breaks represented by contenteditable block elements"**
   - Input: `element('DIV', [element('DIV', [text('first')]), element('DIV', [text('second')])])`
   - Call: `getTextOffset(div, null, 0).text`
   - Expected: `'first\nsecond'`

3. **"uses data-raw when rendered text differs from source text"**
   - Input: `element('DIV', [element('SPAN', [text('acme')], { raw: '[[projects/acme]]' })])`
   - Call: `getTextOffset(div, null, 0).text`
   - Expected: `'[[projects/acme]]'`

4. **"counts data-raw length when selection is after a preview link"**
   - Input: DOM tree with text('open '), SPAN with data-raw and text('acme'), text(' done') in a DIV
   - Call: `getTextOffset(line, line, 2).offset`
   - Expected: `'open [[projects/acme]]'.length` (which is 23)

5. **"maps selection inside preview text into the raw token span"**
   - Input: Same DOM tree as assertion 4
   - Call: `getTextOffset(line, label, 2).offset`
   - Expected: `'open [['.length` (which is 7)

**Helper functions for building DOM trees in happy-dom:**
```typescript
function text(value: string): Text {
  return document.createTextNode(value)
}

function element(
  nodeName: string,
  children: Node[] = [],
  dataset: Record<string, string> = {},
): HTMLElement {
  const el = document.createElement(nodeName)
  children.forEach((child) => {
    el.appendChild(child)
  })
  Object.entries(dataset).forEach(([key, value]) => {
    el.dataset[key] = value
  })
  return el
}
```
